# Sweepstake

WC2026 Sweepstake Tracker — a web app tracking a FIFA World Cup 2026 sweepstake between 26 players, deployed to GitHub Pages. No build step (React 18 + Babel via CDN).

Live app: https://joshmacklin1.github.io/sweepstake/
